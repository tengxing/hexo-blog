Docker安装Tomcat部署应用.md

发表于 2018-05-02

前言

使用docker部署一个应用tomcat，安装见基于Ubuntu安装docker

安装tomcat

安装镜像

搜索tomcat镜像文件信息：

下载想要的版本，默认拉取最新的

启动

在浏览器里输入http://ip:8080验证tomcat是否启动成功

部署应用

部署应用和平时部署一样，都要把war包放在webapps目录下面。有两种方式

  • 挂载目录
  • 拷贝到webapps

挂载目录

  • –privileged=true 授予docker挂载的权限
  • /home/ubuntu/apple/webapps宿主主机目录
  • /usr/local/tomcat/webapps docker容器tomcat目录
  • -p 端口映射
  • -d 后台挂起
  • –name 容器名称

如果报错:已经有应用了，无法启动。docker rm tomcat

拷贝到webapps

  • tomcat 启动时指定的容器名

进入tomcat docker exec -it tomcat /bin/bash

后记

无
