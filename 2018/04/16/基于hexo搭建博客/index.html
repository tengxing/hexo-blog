基于hexo搭建博客.md

发表于 2018-04-16

前言

一直听说用hexo搭建一个拥有自己域名的博客是很酷炫的事情~，在这十一花上半个小时整个hexo博客岂不美哉。

使用Hexo吸引我的是，其简单优雅, 而且风格多变, 适合程序员搭建个人博客,而且支持多平台的搭建，再加上有些node,git基础，分分钟创建出一个高大上的博客。废话少说，开始吧。
特点:

  • Blazing Fast
  • Markdown Support
  • One-Command Deployment
  • Various Plugins

其实真正还是喜欢markdown写文章。

此时，浏览器中打开网址http://localhost:4000，到此安装结束！
